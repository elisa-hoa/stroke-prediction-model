# Stroke Prediction Model using R

## Project Overview

This project builds a stroke prediction model using logistic regression in R.

Main steps:
Data exploration,
Data cleaning,
Missing value handling,
Feature preparation,
Logistic regression modelling,
Prediction generation,
Model evaluation.

## Dataset

Healthcare Stroke Dataset containing demographic and health-related variables

## Tools Used

R,
R Markdown,
tidyverse,
Logistic regression (glm).

## Results 

The analysis identified age, hypertension, heart disease and glucose level as significant predictors of stroke risk.
